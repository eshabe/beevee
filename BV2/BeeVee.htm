Selamat datang di BeeVee : Board Viewer like ECAD Layout Reader Application --ReadMe--!
Terima kasih telah menggunakan program kami, semoga usaha kami membantu Anda tidak sia-sia belaka.

BeeVee adalah program untuk menampilkan layout board (PCBA) dan mencari lokasi komponen, testpin dan jalur dari berbagai sumber data.

Dokumen ini bukanlah panduan untuk menggunakan program tersebut. Panduan penggunaan berada pada dokumen terpisah. 


ISI

  • Instalasi
  • Kemampuan
  • Masalah-masalah
  • Dokumentasi

Instalasi

  1. Salin berkas BeeVee.exe di map sesukamu.
  2. Salin berkas BeeVee.htm di lokasi yang sama.
  3. Salin berkas BeeVee.exe.Manifest di lokasi yang sama juga (tambahan bagi pengguna Windows XP).

Kemampuan

  • Umum:
    - Menampilkan dan mencari lokasi komponen, jalur (trace), testpin (probe/nail/node).
    - Mencari testpin dan jalur dari suatu sinyal atau node.
    - Mencari nama sinyal dari suatu pin.
    - Mencari komponen-komponen yang berhubungan dengan suatu komponen.
    - Melakukan proses pencerminan (flip) dan pemutaran (rotate).
    - Melakukan penggabungan data.
    - Ekpor ke berkas placement.
    - Menyunting komponen.
    - Membandingkan (layout) dua data.
    - Fleksibel dalam Zoom dan pewarnaan.
  • Mengenali data-2 berikut:
    - BeeVee 1.x (.bev)
    - GenRad (.ckt, .tac) + ToolKit (.brd)
    - Agilent ICT 3070 (board, board_xy, fixture)
    - Agilent SJ50 (.plx, .pls)
    - My General XY (TEXT dg format: REFDES{spasi}T|B{spasi}X1{spasi}Y1{spasi}X2{spasi}Y2 -- X2 & Y2 opsional)
    - Placement data (TEXT dg data minimal REFDES,X,Y,ORIENTASI,TOP/BOT)
    - GenCAD (cad)
    - Fabmaster (fab)
    - Unicam (pcb)
    - Pad-PowerPCB (asc)
  • BeeVee mengunakan satuan mils (0.001 inch) dan menganggap bahwa:
    - My Generel XY sudah dalam satuan mils
    - Agilent ICT 3070 dalam 10 mils
    - Agilent AOI SJ50 dalam satuan micron
    - GenRad + Toolkit dalam satuan mils
    - GenCAD dalam satuan mils
    - Fabmaster dalam satuan mils
    - Unicam dalam satuan mils

Masalah-masalah

  • Karena menggunakan std ANSI (yg tidak Thread Safe) dalam mengolah berkas maka program seakan-2 berhenti saat pengolahan berkas.
  • Tidak bisa jalan di Win98. (Setidaknya sampai versi 2.0.5, versi 2.1 belum dicobakan).
  • Tidak mengenali orentasi komponen selain 0,90,180,270
  • Peralihan data BeeVee 2.0 (atau yg lebih lama) ke BeeVee 2.1; yakni data yang bersumber dari mesin tester yang mengikutsertakan nail/node/probe; (mungkin) akan salah dalam penempatan probe-probe yang memiliki nama signal sama. Sebetulnya, dalam BeeVee 2.0 Anda juga tidak dapat membedakan posisi probe-probe tersebut. :)
  • Masalah-2 lain yg belum terdeteksi silahkan beri tahu penulis

Welcome at BeeVee : Board Viewer like ECAD Layout Reader Application --ReadMe--!
Thank you for using this software and hope it help you.

BeeVee is a program to display layout of a board (PCBA) and locate the components/devices, testpins and traces from any sources.

This document is NOT an usage guide. The guide should be in another document.

CONTENTS

  • Installation
  • Abilities
  • Known Problems
  • Documentation

Instalasi

  1. Copy BeeVee.exe file into a folder.
  2. Copy BeeVee.htm file into a same folder.
  3. Copy BeeVee.exe.Manifest file into a same folder (additional for Windows XP users).

 

Abilities

  • General:
    - Displays and locates devices, traces and testpins (probe/nail/node).
    - Locates testpins and traces of a signal or node.
    - Gets signal name of a pin.
    - Locates connected devices of a device.
    - Perform flip and rotate.
    - Merges data.
    - Export to placement file.
    - Edit a device.
    - Compare layout between two files.
    - Flexibility in zooming and coloring.
  • Applicable for data:
    - BeeVee 1.x (.bev)
    - GenRad (.ckt, .tac) + ToolKit (.brd)
    - Agilent ICT 3070 (board, board_xy, fixture)
    - Agilent SJ50 (.plx, .pls)
    - My General XY (TEXT with format: REFDES{spasi}T|B{spasi}X1{spasi}Y1{spasi}X2{spasi}Y2 -- X2 & Y2 opsional)
    - Placement data (TEXT with minimum data contains REFDES,X,Y,ORIENTASI,TOP/BOT)
    - GenCAD (cad)
    - Fabmaster (fab)
    - Unicam (pcb)
    - Pad-PowerPCB (asc)
  • BeeVee use mils (0.001 inches) unit and assumes:
    - My General XY in mils
    - Agilent ICT 3070 in 10 mils
    - Agilent AOI SJ50 in micron
    - GenRad + Toolkit in mils
    - GenCAD in mils
    - Fabmaster in mils
    - Unicam in mils

 

Known Problems

  • This program is based on ANSI C language (not thread safe); it is normal when loadig a file seemly stop.
  • Cannot work in Windows 98. (Confirmed for version 2.0.5, version 2.1 is not tested yet).
  • Does not know any orientations except 0,90,180,270
  • Transfer from BeeVee 2.0 (or before) data to BeeVee 2.1; data from tester machine contains nail/node/probe; (maybe) will get wrong placement of the single signal multi probes. Actually, you can not see where is a probe of the probes of a signal in BeeVee 2.0 :)
  • Other problems, please feedback to the author!
